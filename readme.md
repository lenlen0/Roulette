# 🎲 Roulette des Étudiants

Application PHP/MySQL pour tirer au sort des étudiants par classe et leur attribuer une note sur 20.

## Prérequis

- PHP ≥ 7.4 + MySQL
- Serveur web (Laragon, LAMP, MAMP…)

## Installation

1. Importer `roulette.sql` dans MySQL
2. Renseigner les paramètres dans `_config.php`ss
3. Accéder à `http://localhost/roulette/`

## Utilisation

1. Sélectionner une classe
2. Cliquer sur **🎲 Tirer au sort**
3. Attribuer une note ou passer l'étudiant
4. Répéter jusqu'à ce que toute la classe soit passée
